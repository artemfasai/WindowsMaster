# Claude Code у Windows Master

Claude Code зобов'язаний виконувати [AGENTS.md](AGENTS.md) та локальні
`AGENTS.md`, розташовані ближче до файлів, що змінюються. Цей документ не
послаблює кореневу політику й визначає лише інтеграційну поведінку.

## Обов'язковий порядок роботи

1. Визначити точну ціль, межі та критерії завершення.
2. Перевірити структуру репозиторію, `git status` і чинні інструкції.
3. Не торкатися сторонніх змін та невідстежуваних користувацьких файлів.
4. Підготувати найменшу оборотну зміну, перевірити diff і запустити релевантні
   локальні перевірки.
5. Не вважати успішний запуск команди доказом успіху; перевірити результуючий
   стан окремо.

## Дозволи та безпека

- Не схвалювати автоматично elevation, руйнівні дії, зовнішню публікацію,
  доступ до секретів або зміну засобів захисту.
- Не виконувати remote code, доки джерело, версію, підпис або хеш не перевірено.
- Не надсилати закритий код чи операційні дані до MCP або інших зовнішніх
  сервісів без документованого дозволу.
- Адміністративні тести виконувати лише в явно визначеному ізольованому
  середовищі.

## Звітність

Фінальний звіт повинен містити перелік змін, докази перевірки, ризики,
припущення, відомі обмеження та спосіб відкату, якщо поведінка системи змінилася.
